## Skills GSAP (Cursor/Claude/Codex)

Este repo usa React + Vite + Tailwind. Si quieres añadir animaciones (hero, micro-interacciones, scroll), puedes usar GSAP y aprovechar el paquete de skills oficial de GreenSock para que el agente genere código con buenas prácticas.

- **Repositorio de skills**: [greensock/gsap-skills](https://github.com/greensock/gsap-skills)

### Instalación recomendada (global, reutilizable)

Instala todas las skills para todos los agentes (modo no-interactivo):

```bash
npx -y skills add -g https://github.com/greensock/gsap-skills --all
```

### Comprobar instalación

```bash
npx -y skills ls -g
```

### Qué incluye (resumen)

- `gsap-core`
- `gsap-timeline`
- `gsap-scrolltrigger`
- `gsap-react`
- `gsap-plugins`
- `gsap-utils`
- `gsap-performance`
- `gsap-frameworks`
